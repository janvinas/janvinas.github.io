About

This web app allows you to visualize the current position of trains in the Minecraft server, updated in real time.

Designed and programmed by Jan Viñas (janitus1234@gmail.com). To render the map, the open-source javascript library Leaflet is used.

How to use

To update the map, tap it or press the "Update Map" button on the bottom-left menu.

To get information of a train, press its icon. A popup will appear. To close the popup, click anywhere on the map or select "Close Popup" on the menu. The x button is not currently working.

If you want to automatically update the map every second, use the "Auto Update" and "Stop Auto Update" buttons on the menu.
